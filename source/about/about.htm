ReferenceFinder is a tool for origami design that finds short, efficient, and accurate folding sequences for constructing an arbitrary point or line within a rectangular sheet of paper. Using ReferenceFinder, you can create all of the reference points needed to construct an origami design by folding alone.

ReferenceFinder 4 was written in C++ and developed on Macintosh using Apple Xcode 2.2 for development and wxWidgets (www.wxwidgets.org) for the GUI. Contributors include:

  • Robert J. Lang, main author, Mac and Windows implementation
  • Carlos Furuti, expression parser and GNU/Linux implementation
