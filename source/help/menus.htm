Menus

This section describes each of the menu commands in ReferenceFinder. The menus are:


  • Application Menu
  • File Menu
  • Edit Menu
  • Action Menu
  • Window Menu
  • Help Menu

The commands in each menu and their descriptions follow.

Application Menu

Application->About ReferenceFinder...

Puts up a dialog box that displays brief information about the program, the version, and the terms of usage.

This box shows up the first time you run the application after the splash screen; a checkbox lets you turn off its display thereafter.

The About ReferenceFinder... command shows up under the Help menu on non-Mac platforms.

Application->Preferences

Sets application-wide preferences for ReferenceFinder. See the Preferences Window for a detailed description of the contents and their function.

The Preferences command shows up under the File menu on non-Mac platforms.

Application->Quit

Exit the program. You can quit even if a calculation (database rebuild or statistical analysis) is in progress.

The Quit command shows up under the File menu and is called Exit on non-Mac platforms.

File Menu

File->Export PostScript...

Exports a text file containing PostScript code that prints the diagrams. Due to differences between screen layout and printing settings, the diagram layout in the PostScript file will not exactly match the screen display and/or printouts.

File->Print...

Print the currently-displayed references. The printout will be scaled to the full width of the page, with page breaks, if needed, between folding sequences.

File->Page Margins...

Lets you choose the margins for printing. This item is only present on Macintosh, as on other platforms you can set the margins in the Print Setup... dialog.

File->Print Setup...

Set up the page for printing.

File->Print Preview...

Preview the print image before printing.

Edit Menu

Edit->Undo

Undo the last modification in a text box.

Edit->Cut

Cut the current text selection.

Edit->Copy

Copy the current text selection.

Edit->Paste

Paste the current text selection.

Edit->Clear

Delete the current text selection.

Action Menu

Action->Seek Marks/Lines

Toggle the radio button in the main window between seeking marks and lines. Using the mouse, it's probably quicker and easier to just click on the radio button itself, but this menu item serves as a reminder of the quickest way to toggle between the two: type command-T (control-T on non-Mac platforms).

Action->Get References

Find a set of folding sequences for the reference (point or line) in the main window. This command is automatically performed if you hit the Return or Enter key while typing in one of the text boxes of the main window.

Action->Clear References

Clear the currently-displayed folding sequences. This is useful if you want to keep ReferenceFinder around on-screen at its minimum size.

Action->Rebuild Database

Begin reconstructing the database of marks and lines. Normally you will not need to execute this command -- database rebuild happens automatically when you apply changes to the database settings in the Preferences window -- but if you halted database reconstruction for some reason, you can restart it with this command.

Action->Calculate Statistics

Perform a statistical analysis of the database. This command performs a series of trials, picking random points within the paper, finding the folding sequence that gives the closest mark, and then computing the error between the two. The resulting error is binned. At the end of the analysis, the results are display in two ways: first, the percentage of trials that gave error less than a given amount; second, the error at various percentiles. The results are shown in the main window, however, they are not printable.

This command is useful primarily for comparing the performance of different database settings; you will not ordinarily use it.

Action->Halt Calculation

If the database is under construction, this command is displayed as Halt Rebuild Database and halts the construction of the database. The database will be left in a usable state, but only with those marks and lines calculated so far.

If a statistical analysis is in progress, this command is displayed as Halt Calculate Statistics and halts the analysis. Results will be reported based on the trials accumulated so far.

Window Menu

  • Note: this menu is only present on Mac platforms.

Window->Minimize

Minimize the frontmost window.

Window->Zoom

Zoom out the frontmost window.

Window->Bring All to Front

Bring all windows to the front of the computer display.

Window->ReferenceFinder

Bring the main window to the front of the computer display.

Help Menu

Help->Help

Show these help pages.
